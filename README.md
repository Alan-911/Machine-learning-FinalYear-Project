## Project Goal

Binary classification system to distinguish authentic human speech from AI-generated (synthetic) audio. This project compares three complementary ML paradigms — traditional gradient boosting, deep learning, and probabilistic modeling — to evaluate which best captures the subtle acoustic signatures that separate genuine from manipulated speech.

## Dataset

**ASVspoof 2019** — the gold-standard benchmark for anti-spoofing research.

- Bona fide (genuine human) speech vs. spoofed audio generated by TTS & Voice Conversion systems
- Features extracted: **MFCC** (Mel-Frequency Cepstral Coefficients) and **LFCC** (Linear Frequency Cepstral Coefficients)
- Source: https://huggingface.co/datasets/asvspoof/asvspoof2019

## Models

| Model | Type | Input | Purpose |
|-------|------|-------|---------|
| **GMM** | Probabilistic | MFCC/LFCC vectors | Baseline — acoustic distribution modeling |
| **LightGBM** | Gradient Boosting | Tabular MFCC/LFCC stats | Strong baseline — traditional supervised learning |
| **ResNet/LCNN** | Deep Learning (CNN) | Log-mel spectrograms | State-of-the-art — spatial-temporal pattern learning |

## Evaluation Metrics

- **EER** (Equal Error Rate) — primary metric for anti-spoofing systems
- **Accuracy** — overall correctness
- **F1-score** — balance between precision and recall

## Quick Start

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Download Dataset
```bash
python scripts/download_data.py --output data/raw
```

### 3. Extract Features
```bash
python scripts/extract_features.py --data_dir data/raw --output_dir data/features
```

### 4. Train Models
```bash
# Train all models
python scripts/train.py --model all --features_dir data/features

# Train specific model
python scripts/train.py --model lightgbm --features_dir data/features
python scripts/train.py --model gmm --features_dir data/features
python scripts/train.py --model cnn --features_dir data/features
```

### 5. Evaluate
```bash
python scripts/evaluate.py --features_dir data/features --results_dir results/
```
